Before drafting any client-facing commercial document (proposal, quote, pitch), establish four things FIRST: (1) who reads it (partner vs end client — changes tone, content, and what internal data is exposed), (2) who issues it (white-label vs own brand), (3) language, (4) what must be concealed (vendor names, internal costs, margins).

**Why:** On 2026-06-12 the HireSignal formal proposal was fully built twice — first as Lucca Tech → Kuma Talent in Spanish with purple branding, then rebuilt as Kuma Talent white-label → Canadian end client in English with corporate navy palette (Letter size, PIPEDA clause), because the real audience surfaced only after the first version was done.

**How to apply:** When asked for a proposal/commercial doc, ask these four questions up front (one AskUserQuestion round), even if the request seems fully specified. Internal cost sections never go in the client-facing version. For North American corporate audiences: Letter page size, sober navy/graphite palettes, and mention the applicable privacy law (PIPEDA in Canada).
