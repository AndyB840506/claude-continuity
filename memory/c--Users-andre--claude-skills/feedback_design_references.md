Always check Dribbble, Awwwards, and Pinterest for design references when:
- Building new web pages or UI components
- Redesigning existing pages
- Choosing aesthetic direction, color palettes, or layouts

**Why:** User wants design decisions grounded in current top-quality real-world work, not generic AI defaults. Compare what we have vs. what's trending/best-in-class before generating.

**How to apply:**
1. Search those 3 sites for relevant examples (by industry, style, or component type)
2. Note 2-3 specific design patterns or details worth applying
3. Compare against the current page/component if one exists
4. Then generate or suggest improvements informed by those references
