Mr. Putrid's Den y BTQ son podcasts distintos con identidades visuales distintas. No usar specs de BTQ para MPD ni viceversa.

**Why:** En sesión 2026-06-04, se generaron prompts de tipografía para EP.003 usando estilos inventados en lugar del sistema canónico de MPD establecido en EP.002. El usuario lo corrigió y compartió el prompt de referencia.

**How to apply:** Antes de generar cualquier prompt de artwork para MPD, leer la sección "Sistema tipográfico" en [[mrputridsden-project]]. Para BTQ, leer [[btq-artwork-flow]]. Nunca improvisar specs tipográficos — siempre usar el sistema confirmado del episodio anterior.
